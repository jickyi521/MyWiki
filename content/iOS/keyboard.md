# keyboard#

在一个界面里，如有需要输入的的TextField。则需要考虑以下几点:

>
	1:点击其他空白区域，软键盘应消失
	2:多个TextField 应支持联动，且不同类型的如 Text, Number 软键盘切换
	3: 软键盘弹出是否遮住本应可显示区域
	4: 最后一项可以done，直接触发事件
